# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-15

### Added

- Initial release of AgriConnect platform
- Farmer Dashboard with weather integration and earnings tracking
- Buyer Dashboard with order management
- Marketplace for buying and selling agricultural products
- Crop Recommendation Engine based on soil and weather data
- Disease Detection and Management system
- Crop Calendar for seasonal planning
- Government Schemes information portal
- AI-powered ChatBot for farmer assistance
- Smart Farming tools and analytics
- Multi-language support (English, Hindi, Tamil, Telugu, Kannada, Bengali, Marathi)
- User authentication and role-based access (Farmer/Buyer)
- Logistics and order tracking
- Payment integration
- Profile management
- Modern UI with Radix UI components
- Responsive design for mobile and desktop

### Features

- **Frontend**: React 18 + Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT-based auth with bcrypt
- **UI Components**: Radix UI, Lucide Icons, Recharts
- **Maps**: Leaflet integration for location services

### Security

- Password hashing with bcryptjs
- JWT token authentication
- Protected API routes
- Environment variable configuration
